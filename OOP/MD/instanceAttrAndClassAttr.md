# 实例属性和类属性
### 实例属性就是通过实例创建的属性（类内通过self，类外通过类名），类属性就是在类内部直接创建的属性，通过类名加点语法访问
实例：
```commandline
class Student(object):
    count = 0   # 类属性

    def __init__(self, name):
        self.name = name    # 实例属性
        Student.count += 1
```
### 若存在和类属性同名的实例属性，则类只能使用类属性，实例只能使用实例属性
### 若不存在和类同名的实例属性，则实例可以使用类属性（所有实例使用的同一个类属性）
